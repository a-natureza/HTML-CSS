Lista de Interesses:

  • Front-end: Refere-se ao desenvolvimento da parte da aplicação web com a qual os usuários interagem diretamente. Inclui o layout, design, e comportamento interativo visto nos navegadores.

  • Design UI: Design de interface do usuário foca em melhorar a apresentação e a interatividade de um aplicativo ou site.

  • UX Research: Pesquisa de experiência do usuário, um aspecto essencial para entender as necessidades e comportamentos dos usuários para criar produtos mais eficazes e agradáveis.

  • JavaScript: Uma linguagem de programação essencial para adicionar funcionalidades interativas a páginas web.

  • Frameworks modernos como React: Frameworks de JavaScript, como o React, ajudam a construir interfaces de usuário de forma eficiente e com código reutilizável.


Lista de Passo a Passo:

  1. Aprender HTML e CSS: O primeiro passo para entrar no desenvolvimento web, envolvendo a aprendizagem das linguagens que estruturam e estilizam páginas na web.

  2. Aprender sobre responsividade: Este passo envolve entender como fazer websites que funcionam bem em uma variedade de dispositivos e tamanhos de tela.

  3. Aprender a usar o Figma: Figma é uma ferramenta de design gráfico focada em interfaces de usuário, que facilita a colaboração entre designers e desenvolvedores.

  4. Praticar JavaScript todos os dias: A prática constante é crucial para dominar JavaScript, permitindo a implementação de funcionalidades complexas em websites.

  5. Construir projetos usando React: Aplicar conhecimentos de React em projetos práticos ajuda a solidificar o aprendizado e mostrar competência no uso deste popular framework.
